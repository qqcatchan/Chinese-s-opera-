# AI Immersive Trainer 系統

## 🎭 系統簡介
AI Immersive Trainer 系統係一個 **沉浸式 AI 導師平台**，支援：
- 粵劇唱腔、舞步、武打
- 語言學習（粵語、普通話、英文）
- 運動訓練（舞蹈、健身、武打）
- 健康管理（心率、呼吸、放鬆）

系統提供 **場景轉換、角色互換、音樂同步表演、分數計算演算法、自動化除錯**，並且支援 **GitHub Actions CI/CD** 自動化部署。

---

## 🧩 系統架構
- **[場景轉換](ca://s?q=VR_場景切換功能)**：舞台、教室、運動場、健康室隨時切換。
- **[角色互換](ca://s?q=VR_角色互換功能)**：導師 ↔ 學員 ↔ 表演者。
- **[音樂同步表演](ca://s?q=VR_音樂同步表演)**：口型、舞步、武打全部跟音樂拍子。
- **[課程模組](ca://s?q=AI_課程模組設計)**：粵劇、語言、運動、健康，未來可擴展。
- **[分數計算](ca://s?q=AI_分數演算法檢查)**：根據準確度自動評分。
- **[自動化除錯](ca://s?q=AI_自動化除錯系統)**：即時檢查 Animator、場景、模型、音樂、分數演算法。
