# Demo Recording Script & Narration

Use this script when recording a demo video or doing a live walkthrough. Each step includes suggested narration and timing.

1. Intro (10s)
- Narration: "Hello, this is the Chinese Opera Virtual Tutor mobile demo. We'll show pose detection, vocal feedback, and live peer-to-peer interaction on mobile devices."

2. System overview (10s)
- Narration: "The demo runs in the mobile browser and uses MediaPipe for pose detection, Web Audio for vocal analysis, and WebRTC for live peer-to-peer sessions."

3. Camera & Microphone Preview (15s)
- Action: Open demo URL on phone, allow camera & mic
- Narration: "Here we grant camera and microphone permissions — the app shows a live preview."

4. Vocal feedback (15s)
- Action: Speak into the phone, show volume meter
- Narration: "The voice module analyzes microphone input and displays a real-time volume meter and pitch feedback."

5. Pose detection & correction (30s)
- Action: Move head/torso/arms; show overlay and advice
- Narration: "The pose module overlays detected landmarks and provides simple corrective advice — for example, 'Torso tilted. Stand upright.'"

6. Start Peer-to-Peer session (60s)
- Action: On two devices, join the same room, start P2P
- Narration: "Now we join a room from two devices and start a peer-to-peer session with video and audio — useful for remote tutoring and synchronized practice."

7. Chat and interaction (20s)
- Action: Send chat messages, show remote video
- Narration: "Participants can exchange messages and see each other's video in real-time."

8. Wrap-up and next steps (10s)
- Narration: "This demo shows how the platform supports mobile tutoring for singing, movement, and cultural instruction. Next steps: deploy to cloud, add curriculum, and tune AI models."
