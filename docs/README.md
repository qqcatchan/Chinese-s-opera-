# GitHub Pages for Mobile Demo

This repo includes a static single-page demo suitable for mobile (iPhone) under docs/. To publish via GitHub Pages:

1. Open the repository page in GitHub (mobile browser) and go to Settings → Pages.
2. Under 'Source', choose 'main' branch and folder '/docs'. Save.
3. After a minute, your site will be available at:
   https://qqcatchan.github.io/Chinese-s-opera-

Notes for iPhone testing
- Open the Pages URL in Safari (HTTPS). Allow camera & microphone when prompted.
- For VR Cardboard viewing: open the page, allow camera, then place phone in headset — the page will still render the camera preview and overlays.

If you want me to enable Pages by creating a PR or by doing additional commits, tell me — otherwise enable via the Settings on GitHub as described above.
