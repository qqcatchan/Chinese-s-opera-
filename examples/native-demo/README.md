# React Native (Expo) Native Demo

This folder contains a minimal Expo app example showing how to access the camera and microphone. It is a starting point for integrating tfjs-react-native or TFLite models.

Steps:
1. Install Expo CLI: npm install -g expo-cli
2. cd examples/native-demo
3. npm install
4. expo start

See App.js for a minimal camera preview example.
